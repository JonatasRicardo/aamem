# aamém

Base inicial do aamém, um criador de minisites para igrejas.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- shadcn/ui
- Storybook

## Marca

- Fonte: Adamina
- Cores:
  - Indigo: `#231169`
  - Cocoa: `#2b1d1d`
  - Lavender: `#5a527c`
  - Rose: `#9e6c6c`
  - White: `#ffffff`

## Comandos

```bash
npm run dev
npm run build
npm run lint
npm run storybook
npm run build-storybook
```

O logo principal está em `public/brand/aamem-logo.png`.
